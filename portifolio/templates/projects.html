1. To-Do List Website

Technologies Used: Django, HTML, CSS, SQlite

Project Overview: The To-Do List Website is a simple yet powerful task management tool designed to help users track their daily activities and stay organized. Users can easily create, delete, and mark tasks as completed.

Key Features:
  • Task Management: Create and delete tasks with priority and category tags.
  • Responsive Design: Optimized for desktop, tablet, and mobile devices.
  • Real-Time Sync: SQlite syncs tasks to ensure persistence and real-time updates.
  • Search Functionality: Find tasks quickly by title, category
Challenges Faced:
  • Designing a clean, user-friendly UI/UX.
  • Managing dynamic task data with SQlite.
  • Ensuring smooth authorization session management.
2. Social Service App (Android Studio)

Technologies Used: Java, Android Studio, Firebase, Google Maps API

Project Overview: The Social Service App is an Android-based application that helps users find social service organizations in their locality.

Key Features:
  • User-Friendly Interface: Simple and intuitive UI for easy navigation.
  • Service Categories: Filter services by type such as food, money, clothes, etc.
  • Firebase Backend: Real-time data syncing and authentication with Firebase.
  • Push Notifications: Notifications for new or important updates on social services.
Challenges Faced:
  • Integrating Google Maps for accurate location services.
  • Firebase Authentication and secure data management.
  • Ensuring real-time data synchronization with Firebase.
